VATSIM FLIGHT STRIP PRINTER - QUICK START
==========================================

PROJECT LOCATION:
C:\Users\maxlk\Desktop\vatsim-flight-strip-printer

BEFORE YOU START:
1. Install Node.js from https://nodejs.org/ (LTS version)
2. Connect your POS-5890K printer via USB
3. Ensure printer is powered on with paper loaded

INSTALLATION:
Open PowerShell in the project folder and run:
> npm run install:all

RUNNING THE APPLICATION:
Development mode (recommended for testing):
> npm run dev

Then open browser to: http://localhost:5173

Production mode:
> npm run build
> npm start

Then open browser to: http://localhost:3000

KEY FEATURES:
- Auto-refreshes VATSIM data every 15 seconds
- Filter flights by airport (default: ESGG)
- Print strips remotely from any device on your network
- SVG-based customizable flight strip template
- Text fallback printing if image rendering fails

NETWORK ACCESS:
1. Find your IP: ipconfig (in PowerShell)
2. Access from other devices: http://<YOUR_IP>:5173 (dev) or :3000 (prod)
3. Allow firewall access if prompted

TROUBLESHOOTING:
- Printer not detected? Check USB connection and drivers
- VATSIM data not loading? Check internet connection
- Build errors? Install: npm install --global windows-build-tools
